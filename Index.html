Publish Your Website

Turn your HTML/CSS website into a live link you can share with the world!

Follow these steps to get your Dev with Henok website online

1Save Your Code

Save your HTML file as index.html on your computer.

Right-click → Save As → index.html

2Choose a Hosting Platform

Select a platform to host your website for free:

  • GitHub Pages (developer-friendly)
  • Netlify (easy drag-and-drop)
  • Vercel (excellent for web apps)
  • GitLab Pages (alternative to GitHub)

Step-by-Step Guide for GitHub Pages

1Create GitHub Account

Go to github.com and create a free account if you don't have one.

2Create New Repository

Click on "+" → "New repository"

Name it username.github.io (replace "username" with your GitHub username)

3Upload Your Files

Drag and drop your index.html file into the repository.

You can also upload other files (CSS, images) this way.

4Visit Your Website

Go to https://username.github.io

Your website should now be live! It may take a few minutes to appear.

Your Website Link Will Be:

https://yourusername.github.io
